## About

Quick Notes is a minimal, browser-first note-taking app focused on simplicity and ease of use. It provides a clean, responsive interface so you can quickly create, edit, and remove notes without unnecessary complexity.

## Features

- Lightweight and easy to use
- Clean, responsive styling
- Works as a static web app (no backend required)

## Installation

No build step is required. You can run Quick Notes in one of the following ways:

- Open `index.html` directly in your browser.
- Serve the project folder using a simple static server. For example, with Python 3:

  python -m http.server 8000

  Then open http://localhost:8000 in your browser.

## Usage

- Create a new note using the UI controls.
- Edit or delete existing notes as needed.
- Notes are stored locally in your browser (no server needed).
